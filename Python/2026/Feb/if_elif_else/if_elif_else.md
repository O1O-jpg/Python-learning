# `if elif else`
某些场景下，判断条件不止一个，可能有多个。
这种需求能用Python实现吗?

`if elif else` 语句可以实现
## 格式
```py
if <判断条件1>:
    <判断条件1满足时执行的语句1>
    <判断条件1满足时执行的语句2>
    <判断条件1满足时执行的语句3>
    ...
elif <判断条件2>:
    <判断条件2满足时执行的语句1>
    <判断条件2满足时执行的语句2>
    <判断条件2满足时执行的语句3>
    ...
...
elif <判断条件N>:
    <判断条件3满足时执行的语句1>
    <判断条件3满足时执行的语句2>
    <判断条件3满足时执行的语句3>
    ...
else
    <判断条件均不满足时执行的语句1>
    <判断条件均不满足时执行的语句2>
    <判断条件均不满足时执行的语句3>
    ...
```
**示例一:**
```py
print("欢迎来到Meow餐厅！")
high = int(input("请输入您的身高（厘米）："))
level = int(input("请输入您的会员等级（1-10）："))

if high <= 120:
    print("您的升高小于120cm,可以免费用餐！")
elif level >= 6:
    print("您的会员等级大于等于6级，可以免费用餐！")
else:
    print("很抱歉，您需要支付餐费。")

print("祝您用餐愉快！")
```

**输出一:**
```
欢迎来到Meow餐厅！
请输入您的身高（厘米）：180
请输入您的会员等级（1-10）：6
您的会员等级大于等于6级，可以免费用餐！
祝您用餐愉快！
```
> 满足*等级条件*,可以免费用餐

**输出二:**
```
欢迎来到Meow餐厅！
请输入您的身高（厘米）：121
请输入您的会员等级（1-10）：5
很抱歉，您需要支付餐费。
祝您用餐愉快！
```
> 不满足*任何条件*,需要付费

## 注意点
### `elif`重复使用
**示例:**
```py
# 示例二
print("欢迎来到Meow餐厅！")
high = int(input("请输入您的身高（厘米）："))
level = int(input("请输入您的会员等级（1-10）："))
day = int(input("输入今天的日期:"))

if high <= 120:
    print("您的升高小于120cm,可以免费用餐！")
elif level >= 6:
    print("您的会员等级大于等于6级，可以免费用餐！")
elif day == 1:
    print("Meow恭喜,今天是每月的第一天，您可以免费用餐！")
else:
    print("很抱歉，您需要支付餐费。")

print("祝您用餐愉快！")
```
**输出:**
```
欢迎来到Meow餐厅！
请输入您的身高（厘米）：180
请输入您的会员等级（1-10）：1
输入今天的日期:1
Meow恭喜,今天是每月的第一天，您可以免费用餐！
祝您用餐愉快！
```

> `elif`语句可以写多个

### 判断顺序
判断是互斥且有顺序的。
* 满足<判断条件1>将不会理会<判断条件2>和<判断条件3>
* 满足<判断条件2>，将不会理会<判断条件3>
* <判断条件1><判断条件2><判断条件3>均不满足，进入else
* else也可以省略不写，效果等同个独立的if判断

### 精简写法
**示例:**
```py
print("欢迎来到Meow餐厅！")

if int(input("请输入您的身高（厘米）：")) <= 120:
    print("您的升高小于120cm,可以免费用餐！")
elif int(input("请输入您的会员等级（1-10）：")) >= 6:
    print("您的会员等级大于等于6级，可以免费用餐！")
elif int(input("输入今天的日期:")):
    print("Meow恭喜,今天是每月的第一天，您可以免费用餐！")
else:
    print("很抱歉，您需要支付餐费。")
```
> 直接判断input的输入,而不新增变量,用以节省代码量

**输出:**
```
欢迎来到Meow餐厅！
请输入您的身高（厘米）：180
请输入您的会员等级（1-10）：1
输入今天的日期:1
Meow恭喜,今天是每月的第一天，您可以免费用餐！
```

## 总结
### `if elif else`语句的作用
可以完成多个条件的判断
### 注意点
* elif可以写多个
* 判断是互斥且有序的，上一个满足后面的就不会判断了
* 可以在条件判断中，直接写input语句，节省代码量
