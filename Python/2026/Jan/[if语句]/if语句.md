# if语句
## 格式
```py
if <要判断的条件>:
    条件成立时要做的事情
```
>[!note]
> 与C/C++不同的是,if不是通过{}来划定范围
> 
> 而是通过:与缩进来表示



**示例:**
```py
a = 10
if(a == 10):
    print("a等于10")
```
**输出:**
```
a等于10
```

**示例二:**
```py

# 示例2
age = input("请输入你的年龄：")

if int(age) >= 18:
    print("你是成年人。")
    print("即将步入社会了")

print("时间过得好快")
```
**输出:**
```
请输入你的年龄：88
你是成年人。  
即将步入社会了
时间过得好快 
```
**输出二:**
```
请输入你的年龄：13
时间过得好快
```
> `print("时间过得好快")`在判断之外,所以不受影响

## 注意点
判断语句的结果，必须是布尔类型True或False
True会执行if内的代码语句
False则不会执行

## 总结
### if语句的注意事项
* 判断条件的结果一定要是布尔类型
* 不要忘记判断条件后的：冒号
* 归属于if语句的代码块，需在前方填充4个空格缩进
