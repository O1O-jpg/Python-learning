# 变量
## 什么是变量
在程序运行时，能储存计算结果或能表示值的抽象概念。简单的说，变量就是在程序运行时，记录数据用的.
## 变量的定义格式
`<变量的名称> = <变量的值>`
### 变量名
每个变量都会有自己的名称,称之为*变量名*,也就是变量本身
### 变量值
每一个变量都有自己存储的值,称之为*变量值*
### 赋值
"="表示*赋值*.表示将等号右侧的值,赋予左侧的变量
### 四则运算
| 现实表示 | 代码表示 |
| :------: | :------: |
|    加    |    +     |
|    减    |    -     |
|    乘    |    *     |
|    除    |    /     |

## 总结
### 变量的概念及作用
变量就是在程序运行时，记录数据用的
### 变量的定义格式
`<变量的名称> = <变量的值>`
### 变量的特征
变量的值可以改变
