# 字面量
## 什么是字面量
字面量：在代码中，被写下来的的固定的值，称之为字面量
## Python中常用的六种数据类型
* **数字(Number)**
  * **整数**
  * **浮点数**
* **字符串(String)**
* 列表(List)
* 元组(Tuple)
* 集合(Set)
* 字典(Dictionary)
### 字符串
字符串（string），又称文本，是由任意数量的字符如中文、英文、各类符号、数字等组成。所以叫做字符的串.

**例子:**
> `"你好!"` - 中文字符串
> `"123"` - 数字形式的字符串

>[!note]
>在Python中,字符串需要使用引号包起来.
>
>被双引号包裹的都是字符串

### 如何在代码中表达

|  类型  |   写法    |          说明          |
| :----: | :-------: | :--------------------: |
|  整数  |  666,123  |     与现实写法一致     |
| 浮点数 | 66.6,12.3 |     与现实写法一致     |
| 字符串 | "代理人"  | 程序中需要加双引号表示 |

**程序表示:**
``` Python
print(123) # 数字
print(12.34) # 小数
print("HelloWorld") # 字符串
```

## 总结
### 字面量的含义
代码中，被写在代码中的固定的值，称之为字面量
### 常见的字面量类型
*整数*,*浮点数*,*字符串*
### 如何基于`print()`输出字面量
**格式:**
>`print(字面量)`
