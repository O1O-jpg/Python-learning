# 数据类型
在学习字面量的时候，我们了解到：数据是有类型的。

## 如何验证数据类型
使用`type()`语句来查看变量的数据类型

**语法:**
`type(被查看的数据)`

使用`print(type(<数据>))`能够直接打印该数据的数据类型

```
示例:

print(type("代理人"))

输出:

<class 'str'>

str是string的缩写
```
也可以通过存储到变量,在打印的方式
```
示例:

str_type = type("代理人")
print(str_type)

输出:

<class 'str'>
```

### `type()`语句使用方法
`type()`不仅可以用来查看数据类型,也可以用来查看变量中存储的数据类型
```
示例:

name = "代理人"
name_type = type(name)
print(name_type)

输出:

<class 'str'>
```

## 变量是否存在类型一说
我们通过type（变量）可以输出类型，这是查看变量的类型还是数据的类型？

查看的是：变量存储的数据的类型。因为，变量无类型，但是它存储的数据有。

我们可能会说：字符串变量
但要知道，不是变量是字符串，而是它存储了：字符串

## 总结
### 什么语句可以查看变量的数据类型
`type()`语句
### 变量有没有类型
没有，字符串变量表示变量存储了字符串而不是表示变量就是字符串
